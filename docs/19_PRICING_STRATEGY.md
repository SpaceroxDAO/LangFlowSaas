# Pricing Strategy: Education-First Differentiation

**Last Updated**: 2026-01-25
**Status**: Active Strategy
**Author**: Adam (Founder) + Claude Code Analysis

## Executive Summary

Teach Charlie AI's core differentiation is **not** the AI agent builder itself (commodity) but the **missions-based education system** that transforms users from "AI-curious" to "AI-competent" while building real agents. This document captures the pricing strategy derived from this insight.

---

## The Core Insight

### What Competitors Sell: Tools
Competitors (Botpress, Voiceflow, Zapier Interfaces) sell AI agent building tools. Users must figure out how to use them independently. High churn from users who "didn't get it."

### What Teach Charlie Sells: Capability Transformation

Through guided missions, users:
1. **Learn** AI fundamentals step-by-step
2. **Build** real working agents immediately
3. **Understand** how their agents work (can update/extend)

This creates a **Triple Value Stack**:

| Value Layer | Description | User Outcome |
|-------------|-------------|--------------|
| **Immediate Solution Value** | Working agent | "I have a customer support bot" |
| **Learning Value** | Understanding AI concepts | "I know how prompts work" |
| **Long-Term Capability Value** | Ability to iterate | "I can update this myself" |

### Why This Matters for Pricing

- **Individual users** become more productive in AI capabilities
- **Business teams** multiply their AI productivity across employees
- **Stickiness** increases because users invested learning effort
- **Churn decreases** because users understand what they built

---

## Value Metric Analysis

### Rejected: Usage-Based (Credits/Runs)
- Confusing terminology (what is a "credit"?)
- Doesn't align with education value
- Commoditizes the offering

### Adopted: Learning Outcomes
- **Primary Metric**: Missions Completed
- **Secondary Metrics**: Agents Built, Canvas Time, Capability Level
- Aligns price with transformation delivered

## Tier Structure

### FREE: "AI Foundations"
**Transformation**: AI-Curious → AI-Aware

| Aspect | Details |
|--------|---------|
| **Target User** | Workshop attendees, curious learners |
| **Learning Access** | 3 Starter Missions (basics only) |
| **Building Limits** | 1 Live Agent, 1,000 runs/month |
| **Canvas Access** | Preview only (read-only) |
| **Outcome** | "I understand AI agents and built my first one" |

### INDIVIDUAL: "AI Mastery" - $19/mo ($15/mo yearly)
**Transformation**: AI-Aware → AI-Competent

| Aspect | Details |
|--------|---------|
| **Target User** | Small business owners, creators, solopreneurs |
| **Learning Access** | Full Mission Library (20+ missions) |
| **Building Limits** | Unlimited Agents, Unlimited Runs |
| **Canvas Access** | Full access |
| **Extra** | AI Academy, Community Access, Export/Embed |
| **Outcome** | "I can build any AI agent I need" |

### BUSINESS: "AI-Enabled Teams" - Custom
**Transformation**: AI-Competent → AI-Enabled Organization

| Aspect | Details |
|--------|---------|
| **Target User** | Teams, organizations, enterprises |
| **Learning Access** | Full + Custom Learning Paths |
| **Building Limits** | Unlimited everything |
| **Extra** | Live Workshops, Team Progress Tracking, SSO, Audit Logs |
| **Outcome** | "Our whole team can build AI agents" |

## Competitive Positioning

### Price Anchors (What We're Compared Against)

| Alternative | Cost | Time Investment | Outcome |
|-------------|------|-----------------|---------|
| Online AI course | $200-500 | 20+ hours | Theory only, no product |
| Hire consultant | $2,000-5,000 | One-time | Get agent, learn nothing |
| DIY (ChatGPT + tutorials) | $20/mo | 100+ hours | Maybe works, high frustration |
| **Teach Charlie** | **$228/year** | **Ongoing** | **Learn + Build + Maintain** |

### Why $19/mo is Correct
- Below most AI builder tools ($40-89/mo)
- Far below course + tool separately ($200-500 + $20-50/mo)
- Premium enough to signal value
- Room to increase after validation ($24-29/mo)

### Competitive Moat

| Moat Layer | Description |
|------------|-------------|
| **Founder DNA** | 10 years teaching non-technical users |
| **Workshop Ecosystem** | In-person training feeds platform |
| **Mission Library** | Takes time to create quality guided builds |
| **Community** | Learners help each other (network effect) |

| Metric | Description | Target |
|--------|-------------|--------|
