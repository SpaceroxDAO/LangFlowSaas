# Subagents Library Pack

Catalog of 20+ specialized agents for various domains.

## Installation

```bash
./scripts/enable-pack.sh subagents-library
```

## Usage

Browse the `packs/subagents-library/catalog/` directory for available agents.
Copy specific agents to `.claude/agents/` as needed.
