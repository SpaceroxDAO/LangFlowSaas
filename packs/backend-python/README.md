# Backend Python Pack

Python backend development tools for FastAPI, Flask, Django, etc.

## What This Adds

- API endpoint generation
- Virtual environment setup helpers
- Python backend architecture agent
- Testing utilities

## Installation

```bash
./scripts/enable-pack.sh backend-python
```

## Stack Support

- FastAPI
- Flask
- Django
- Generic Python services
